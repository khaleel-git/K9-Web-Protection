To add a Web site to a list, enter the URL and click Add to List. To delete a Web site from a list, click the red icon next to its URL. URLs that display a time, like '(12 min.)' have been temporarily allowed and will be automatically removed from the list when the time goes to zero.

Blacklist Without Password allows you to add sites to the Always Block list without needing to enter the administrator password (helpful for self-moderators). When selected a link will be given which can be bookmarked to quickly add any visited site to the list.
