Welcome to K10 Web Protection

This installer will set up K10 Web Protection on your Mac.

What gets installed:

  • K10 Web Protection app in your Applications folder
  • Background service that starts automatically at login
  • Watchdog that keeps the filter running
  • Firewall rules that prevent browsers from bypassing the filter

K10 Web Protection filters web content by running a local proxy on your Mac. No data leaves your device.

Administrator password is required. macOS 12.0 or later.
